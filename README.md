# UK Personal Investment Assistant

Interactive single page web app for UK based personal investing.

It helps you:
- See a breakdown of tax, NI, student loans, and pension contributions.
- Visualise net worth projections over time.
- Explore simple scenario comparisons for different savings and pension contribution levels.
- Get indicative allocation ranges based on risk tolerance.

**Tech stack**

- HTML, CSS, vanilla JavaScript
- Chart.js for charts
- No frameworks or build tools
